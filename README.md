# 命理测算引擎 (Fortune Teller)

基于传统命理经典著作的 Python 算命工具，涵盖八字四柱、六爻纳甲、梅花易数、周易六十四卦，纯 Python 实现，无需外部依赖。

## 功能

- **八字四柱排盘** — 年柱、月柱、日柱、时柱，含天干、地支、五行、十神、纳音
- **地支藏干** — 每柱地支所藏天干
- **五行强弱分析** — 五行分布统计、日主分析、喜用神推断
- **大运排盘** — 十年一大运，含起运年龄、阴阳顺逆
- **六爻纳甲占卜** — 铜钱起卦、纳甲装卦、六亲六兽、用神选取、月建日辰断卦
- **梅花易数起卦** — 时间起卦法，得本卦、变卦、动爻、体用生克
- **周易六十四卦解读** — 卦名、卦辞、吉凶断语
- **综合批断** — 日主性格、五行平衡、流年大运简评

## 使用方法

### 八字排盘

```bash
python divination.py 年 月 日 [时] [分] [性别]
```

**示例：**
```bash
# 2004 年 8 月 1 日 18:00，女性
python divination.py 2004 8 1 18 0 女
```

### 六爻纳甲占卜

```bash
python divination.py --liuyao [占问事项]
```

**示例：**
```bash
# 自动摇卦占问
python divination.py --liuyao 丢了钱包能找回来吗

# 交互式铜钱起卦
python divination.py --liuyao --interactive
```

### 梅花易数占卜

```bash
python divination.py --meihua 年 月 日 [时] [分]
```

**示例：**
```bash
python divination.py --meihua 2026 6 22
```

## 算法说明

- **节气计算**：基于开普勒方程迭代求解太阳黄经，精确定位二十四节气
- **日柱干支**：采用万年历基准推算，1900-2100 年精确
- **月柱起法**：依五虎遁（年上起月），考虑节气分界和立春前后
- **时柱起法**：依五鼠遁（日上起时）
- **大运排盘**：阳男阴女顺行、阴男阳女逆行，精确到岁
- **梅花易数**：时间起卦法，年月日时取数，动爻取变
